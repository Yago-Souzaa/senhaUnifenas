# **App Name**: SenhaFacil

## Core Features:

- Importar Senhas: Permitir a importação de dados de senha de uma planilha com as colunas: NOME, IP, LOGIN, SENHA, FUNCAO, ACESSO, VERSÃO.
- Armazenamento de Senhas: Armazenar nomes de usuário e senhas de forma segura localmente (sem armazenamento em nuvem).
- Copiar para Área de Transferência: Habilitar a cópia de credenciais para a área de transferência com um clique.
- Pesquisar e Filtrar: Pesquisar e filtrar entradas de senha com base em palavras-chave ou categorias.
- Gerador de Senhas: Gerar uma senha aleatória segura com base em critérios personalizáveis.

## Style Guidelines:

- Cor primária: Índigo profundo (#4B0082) para uma sensação de segurança e sofisticação.
- Cor de fundo: Cinza claro (#F0F0F0) para fornecer um pano de fundo limpo e neutro.
- Cor de destaque: Teal (#008080) para destacar ações importantes e elementos interativos.
- Fonte do título: 'Inter' sans-serif para títulos e cabeçalhos limpos, modernos e facilmente legíveis.
- Fonte do corpo: 'PT Sans' sans-serif fornece clareza e funciona bem para textos detalhados ou listagens de senhas.
- Usar ícones simples e claros para representar diferentes funções e tipos de credenciais.
- Implementar animações sutis para feedback do usuário, como uma animação de escala ao copiar uma senha.
